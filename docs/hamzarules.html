A Rules for writing hamzaḧ

hamzaḧ is written in four different ways:

  1. Seated on an ʾalif: أ
  2. Seated on an wāw: ؤ
  3. Seated on an yāʾ: ئ
  4. Unseated: ء

Here are some of notes about writing hamzaḧ in the above four methods:

  • When unseated hamzaḧ is followed by an ʾalif: ءا, the combination of hamzaḧ and ʾalif is conventionally written as آ as long as the ʾalif is not a suffix or part of a suffix. When the ʾalif is a suffix or part of a suffix then we will write ءا, not آ.

  • When unseated hamzaḧ comes between two letters that are joined, then it is written above the line that joins them, for example: خَطِيـَٔةٌ k͟haṭīʾatun. In this word, the yāʾ ي joins to the looped tāʾ ة. So the unseated hamzaḧ is written above the joining line like so ــٔـ.

  • When hamzaḧ is seated on ʾalif, if it has an i-mark, it is written below the ʾalif: إِ. Otherwise, it is written above the ʾalif: أَ, أُ, أْ.

  • When hamzaḧ is seated on yāʾ ئ the dots of the yāʾ are no longer written. Here’s how it will appear in different positions:

    Isolated End Middle Beginnning
    ئ ـئ ـئـ ئـ

So how do we know when to write hamzaḧ unseated and when seated? And how do we choose between its three different seats? There are a set of rules that we need to follow in order to correctly write hamzaḧ. These rules are quite complicated (although they have a basic underlying strategy in them). We will give the complete set of rules here. But for now, you may limit yourself to understanding the rules for hamzaḧ in the beginning of a word. The remaining rules are here for you to refer back to as you come across words with hamzaḧ.

  1. If hamzaḧ occurs in the beginning of a word:
    1. If the hamzaḧ carries a long-ā vowel, it is written unseated followed by an ʾalif and written as آ, for example آمَنَ ʾāmana.
    2. If the hamzaḧ carries any other vowel, it is written seated on an ʾalif, and is marked with the appropriated vowel mark, for example أَسْلَمَ ʾaslama, أُرِيدُ ʾurīdu, إِسْلَامُ ʾislāmu, إِيمَانُ ʾīmānu, أُوخِذَ ʾūk͟hiẕa.
  2. If hamzaḧ occurs in the middle of a word:
    1. If there is a long vowel or semi-vowel before the hamzaḧ:
      1. If the hamzaḧ is after a long-ī vowel or ay semi-vowel, then the hamzao will be written unseated. Examples:
        هَيْـَٔةُ hayʾatu, خَطِيـَٔةُ k͟haṭīʾatu
        مُسِيـُٔونَ musīʾūna, مُسِيـَٔانِ musīʾāni, مُسِيـِٔينَ musīʾīna, مُسِيـَٔيْنِ musīʾayni
        شَيْـُٔهُ s͟hayʾuhu, شَيْـَٔهُ s͟hayʾahu, شَيْـِٔهِ s͟hayʾihi, شَيْـَٔانِ s͟hayʾāni, شَيْـَٔيْنِ s͟hayʾayni
        مَجِيـُٔهُ majīʾuhu, مَجِيـَٔهُ majīʾahu, مَجِيـِٔهِ majīʾihi
      2. If the hamzaḧ is after a long-ū vowel or aw semi-vowel, then:
        If the hamzaḧ has an i-mark it is written seated on yāʾ. Examples: سُوئِهِ sūʾihi, ضَوْئِهِ ḍawʾihi
        Otherwise, the hamzaḧ is written unseated. Examples: سُوءَه sūʾahu, سُوءَانِ sūʾāni, تَوْءَمُ tawʾamu, ضَوْءَهُ ḍawʾahu, ضَوْءَانِ ḍawʾāni, سُوءُهُ sūʾuhu, يَسُوءُونَ yasūʾūna
      3. If the hamzaḧ is after a long ā-vowel, then: If the hamzaḧ has an i-mark it is written seated on yāʾ. Example: سَائِلُ sāʾilu.
        If the hamzaḧ has an u-mark it is written seated on wāw. Example: تَسَاؤُلُ tasāʾulu.
        Otherwise, when the hamzaḧ has an a-mark, it is written unseated. Example: سَاءَلَ sāʾala, قِرَاءَاتُ qirāʾātu.
    2. If the letter before the hamzaḧ has a \(\emptyset\)-mark and is not wāw or yāʾ, then:
      1. If the hamzaḧ was originally at the end of the word, but a suffix has been attached to the word, then the hamzaḧ will be written unseated. Examples: عِبْـَٔانِ ɛibʾāni, عِبْـَٔييْنِ ɛibʾayni, بُطْـَٔهُ buṭʾahu, بُطْـُٔهُ buṭʾuhu, بُطْـِٔهِ buṭʾihi. (انِ, يْنِ, هُ, and هِ are suffixes).
      2. Otherwise, if the hamzaḧ is originally at the middle of the word, then it is written seated on wāw if it has an u-mark and yāʾ if it has an i-mark. Examples: مَسْؤُولُ masʾūlu, تَرْئِيسُ tarʾīsu.
        If the hamzaḧ has an a mark then:
        If it is followed by a long-ā vowel represented by an ʾalif, the hamzaḧ is unseated followed by the ʾalif and the combination is written as آ. Example: مِرْآةُ mirʾātu, ظَمْآنُ ẓamʾānu.
        Otherwise, if there is no ʾalif after the hamzaḧ, the hamzaḧ is written seated on ʾalif. Examples: مَسْأَلَةُ masʾalatu, , مَرْأَةُ marʾatu.
    3. If the hamzaḧ has a \(\emptyset\)-mark, then it is written seated on ʾalif if it has an a-mark, wāw if it has an u-mark, and yāʾ if it has an i-mark. Examples: کَأْسُ kaʾsu, بِئْسَ biʾsa, سُؤْلَکَ suʾlaka.
    4. Otherwise, only if the above conditions are not satisfied, then compare the vowel marks of the hamzaḧ and the letter before it:
      1. If either vowel mark is an i-mark then the hamzaḧ will be written on a yāʾ. Examples: سُئِلَ suʾila, يَئِسَ yaʾisa, مُتَّکِئِينَ muttakiʾīna.
      2. If neither vowel mark is an i-mark, and at least one of the vowel marks is a u-mark, then the hamzaḧ will be written on a wāw. Examples: سُؤَالُ suʾālu, رُؤُوسُ ruʾūsu, لُؤَيُّ luʾayyu.
      3. Otherwise, if both of the vowel marks are a-marks, then:
        If the hamzaḧ is followed by a long-ā vowel represented by an ʾalif, the hamzaḧ is written unseated. Examples: شَنَآنُ s͟hanaʾānu.
        Otherwise the hamzaḧ will be written on an ʾalif. Examples: سَأَلَ saʾala, رَأَىٰ raʾā.
      These set of rules apply even if the hamzaḧ or the previous letter is doubled. Examples رَأَّسَ raʾʾasa يُرَئِّسُ yuraʾʾisu رُئِّسَ ruʾʾisa تَفَؤُّلُ tafaʾʾulu يُبَرِّئُونَ yubarriʾūna يُبَرَّؤُونَ yubarraʾūna.
  3. If hamzaḧ is at the end of a word, disregard the vowel mark on it and consider only the letter before the hamzaḧ.
    1. If there is a long vowel (ā, ī, ū) or a semi-vowel (aw, ay) before it then the hamzaḧ will be written unseated. Examples: دُعَاءُ duɛāʾu, سُوءُ sūʾu, جِيءَ jīʾa, ضَوْءَ ḍawʾa, شَيْءَ s͟hayʾa.
    2. Otherwise, if the previous letter has a \(\emptyset\)-mark, the hamzaḧ will again be unseated. Examples: بُطْءُ buṭʾu, عِبْءُ ɛibʾu, شَطْءُ s͟haṭʾu
    3. Otherwise, if the previous letter is a doubled wāw with an u-mark, the hamzaḧ will again be unseated. Example تَبَوُّءُ tabawwuʾu.
    4. Otherwise, if the previous letter has an:
      1. a-mark, the hamzaḧ is written seated on ʾalif. Example يَهْدَأُ yahdaʾu, مُبْتَدَإِ mubtadaʾi.
      2. i-mark, the hamzaḧ is written seated on yāʾ. Example يُهَدِّئُ yuhaddiʾu, سَيِّئُ sayyiʾu.
      3. u-mark, the hamzaḧ is written seated on wāw. Example بَطُؤَ baṭuʾa.

Some further notes:

  • If hamzaḧ is in the beginning of a word, adding a prefix to the word will not alter the writing of the hamzaḧ. Examples: لِ + أُسْتَاذِ = لِأُسْتَاذِ.

  • If hamzaḧ is at the end of a word, adding a suffix to the word can, in general, alter the writing of the hamzaḧ, except in cases that have already been mentioned above. Examples:
    مُبْتَدَأَ + انِ = مُبْتَدَءَانِ
    دُعَاءُ + هُ = دُعَاؤُهُ
    ضَوْءِ + هِ = ضَوْئِهِ
    بُطْءُ + هُ = بُطْـُٔهُ

  • As we mentioned earlier, when unseated hamzaḧ is followed by an ʾalif which is not a suffix: ءا, the combination of hamzaḧ and ʾalif is conventionally written as آ. However, if the unseated hamzao is doubled and then followed by an alif: ءَّا then it won’t be written as آ. Example: سَـَّٔالُ saʾʾālu.
